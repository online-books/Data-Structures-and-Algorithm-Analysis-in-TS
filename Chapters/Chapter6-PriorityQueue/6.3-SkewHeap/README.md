## 斜堆

斜堆(skew heap)是左式堆的自调节形式。斜堆是具有堆序的二叉树，但是没有对树结构的限制。与左式堆不同的是，对斜堆的右路径没有任何限制，因此所有操作的最坏情形运行时间均为$O(N)$。

斜堆的基本操作也是合并。对于左式堆，在合并过程中递归地对节点的左右子树进行堆序性质检查并交换不满足左式堆性质的节点。但对于斜堆，右路径上所有节点的左右子树都会交换位置。假设要合并图 6-11 中的两个斜堆。
